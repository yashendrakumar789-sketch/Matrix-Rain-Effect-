This Matrix Digital Rain project is a high-performance, visually authentic recreation of the iconic "falling code"
sequence from *The Matrix* film series, built entirely with the HTML5 Canvas API and Vanilla JavaScript. By leveraging a
procedural animation loop, the script dynamically calculates vertical columns based on the browser's width and populates
them with a randomized blend of Katakana and alphanumeric characters. The project achieves its signature "phosphor glow"
and trailing effect not by clearing the screen, but by repeatedly layering a semi-transparent black rectangle over the
canvas, which causes older characters to gradually fade into the background.

Designed with modern web standards in mind, the application is fully responsive, automatically recalculating its grid
and resizing the canvas whenever the window dimensions change. Because it relies on zero external libraries or heavy
frameworks, the codebase is exceptionally lightweight and optimized for smooth performance across all modern browsers.
It serves as both a nostalgic aesthetic piece and a practical demonstration of how simple mathematical logic and
probability-based resets can be used to create complex, organic-looking motion graphics in a web environment.
